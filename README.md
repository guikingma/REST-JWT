# REST-JWT

Exemplo de REST utilizando Django Rest Framework e JWT

## Uso para servidor em http://127.0.0.1:8000

### Produtos:

http://127.0.0.1:8000/loja/produtos/

http://127.0.0.1:8000/loja/produtos/[id]

http://127.0.0.1:8000/loja/produtos/[id]/desc


### Pedidos:

http://127.0.0.1:8000/loja/pedidos/

http://127.0.0.1:8000/loja/pedidos/[id]

http://127.0.0.1:8000/loja/pedidos/[id]/desc
